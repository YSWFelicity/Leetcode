📝 字符频次数组笔记
1️⃣ 常见写法对比
字符集范围	数组大小	下标写法	备注
仅小写字母 a-z	26	cnt[c - 'a']	最省空间、最快；推荐用
大小写混合 / 数字 / 符号	256	cnt[c]	覆盖所有 ASCII 字符
任意 Unicode	Map	map.getOrDefault(c, 0)	适合非 ASCII 或字符集太大时

2️⃣ 代码示例
✅ 仅小写字母：
int[] cnt = new int[26];
for (char c : s1.toCharArray()) {
    cnt[c - 'a']++;
}
比如lc 567
✅ 包含大小写或符号：
int[] cnt = new int[256];
for (char c : s1.toCharArray()) {
    cnt[c]++;
}
比如lc 76
✅ 通用解法（支持 Unicode）：
Map<Character, Integer> map = new HashMap<>();
for (char c : s1.toCharArray()) {
    map.put(c, map.getOrDefault(c, 0) + 1);
}

3️⃣ 记忆口诀
小写 26，ASCII 256，通用用 Map
只要题面明确写了“lowercase letters”，优先用 26，既快又省内存。
